# HASSAN.EXE | Portfolio v4.0

A retro-terminal themed personal portfolio showcasing my projects, experience, and skills as an AI Engineer and Data Scientist. 

## 🚀 Features
* **Terminal Boot Sequence:** Animated BIOS-style loading screen on launch.
* **Retro UI/UX:** Custom crosshair cursor, CRT-style glow effects, and glitch-text hero animations.
* **Interactive Modules:** Code-block styled skill sections and simulated terminal windows for a cohesive developer aesthetic.
* **Fully Responsive:** Adapts cleanly to both mobile and desktop displays without external layout libraries.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom properties, Flexbox/Grid layouts, and CSS keyframe animations (zero external CSS frameworks).
* **Vanilla JS:** Scroll progress tracking, dynamic clock, boot sequence logic, and intersection observers for scroll-reveal animations.
